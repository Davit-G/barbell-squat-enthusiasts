# Unihack 2023 Client App

## Getting Started

### Install with Yarn
```bash
yarn install
```

### ... or install with NPM
```bash
npm install
```

## Running the app

### Yarn
```bash
yarn dev
```

### NPM
```bash
npm run dev
```


## Building the app

### Yarn
```bash
yarn build
```

### NPM
```bash
npm run build
```
